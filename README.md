# 서비스 소개

## **사용자에게 피드백을 제공하여 정확한 운동 자세를 제시**

> 

<br />

## **운동 자세가 걱정되시나요?**

> 모션 인식을 통해 어느 부분에서 자세가 문제가 있는지 표시해줘요

<br />

## **영상말고 다른 방법으로 피드백을 받고 싶나요?**

> 영상이 불편할 수도 있어서 텍스트로 자세하게 설명해줘요

<br />

## ***FitMotion에서 정확한 자세로 운동하고 있는지 확인해봐요*
